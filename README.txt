============================================
=== Cours de Physique (ancien programme) ===
============================================

Voici des fiches de résumé du cours de Physique
(ancien programme). Elle contiennent pour la
plupart les formules importantes, ainsi qu'une
piste pour les reprouver. Vous aurez également
les idées principales du cours, parfois quelques
méthodes...

Si vous le désirez vous pouvez également les
mettre sur votre calculatrice NSpire, et les
consulter au moyen du logiciel mViewer 
(nécessite l'installation de Ndless).
